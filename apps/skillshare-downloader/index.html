Skillshare Downloader

Download Skillshare courses for offline viewing with subtitles

📥 How to Download

  1. Log in to Skillshare in Chrome and make sure you have a Premium subscription
  2. Run the app locally on your computer
  3. Choose auth — browser cookies or upload a cookies.txt file
  4. Paste a course URL and click Preview to see all lessons
  5. Select sessions to download (or select all), enable subtitles if needed
  6. Click Download and wait — videos save directly to your computer

⭐ Features

📂 Course Preview

See all lessons before downloading. Skip what you already have.

🎯 Selective Download

Pick individual lessons or download the whole course at once.

📝 Subtitles

Download English subtitles alongside each video automatically.

⏹ Stop Any Time

Cancel downloads mid-way. Already finished videos stay saved.

🔒 Private Repo

Source code hosted in a private GitHub repository for your use only.

🌐 Browser Cookies

Auto-detect Chrome/Firefox/Edge login — no manual export needed.

🛠️ Quick Start (Local)

Run on your Windows machine (requires Python):

cd G:\skillshare-gui
python app.py

Then open http://localhost:5757 in your browser.
